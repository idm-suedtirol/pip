Markdown Text formatting made easy

Markdown is a simple way to highlight and format text. It provides a minimalistic set of rules that allows to create heading, listings and to give emphasis to your contents.

A lot of content-management-systems provide extensive rich text editors that give a lot of possibilities, but on the other hand cause disambiguity with the rest of the website, resulting in ugly and bad formatted websites.
John Gruber of Daring Fireball created a solution called "Markdown".

To get a better insight and to learn how to use "Markdown", visit this nice tutorial written by Graham Ashton:
http://nestacms.com/docs/creating-content/markdown-cheat-sheet
